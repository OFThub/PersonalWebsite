# PersonalWebsite

Özellikler
+ Blog Sistemi (Rich Text Editor)
+ Proje Showcase
+ Admin Panel
+ Dark/Light Mode
+ Çoklu Dil (TR/EN)
+ Global Search (Cmd+K)
+ Analytics Dashboard
+ SEO Optimized
+ Image Upload (Cloudinary)
+ Responsive Design

Tech Stack

Frontend: Next.js 14, TypeScript, Tailwind CSS
Backend: Next.js API Routes, MongoDB, Mongoose
Auth: JWT + bcrypt
Upload: Cloudinary
Editor: TipTap
Charts: Recharts


Dosya Yapısı
app/
├── api/              # API endpoints
├── admin/            # Admin panel
├── blog/             # Blog pages
├── auth/             # Login/Register
└── page.tsx          # Homepage

components/           # React components
models/               # MongoDB models
lib/                  # Utilities

→ Yayınla!
Proje Ekle
Admin → Projeler → Yeni Ekle
→ Başlık, açıklama, teknolojiler
→ GitHub & Demo URL
→ Kaydet!
Global Search
